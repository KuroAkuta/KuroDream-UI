# 简介
本ui组件适配Vue3+vite
<hr>
 
# 安装
## Node：
<pre v-highlight>
<code>npm install kurodream-ui </code>
</pre>


<hr>


# 导入
## main.ts
<pre v-highlight>
<code>import KuroDream from 'kurodream-ui'

import '../node_modules/kurodream-ui/dist/style.css'

app.use(KuroDream);</code>
</pre>


## vite-env.d.ts
<pre v-highlight>
<code>declare module "kurodream-ui"</code>
</pre>
